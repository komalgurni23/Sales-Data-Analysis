# Sales Data Analysis Report

## Project Overview
This project analyzes a sales dataset using Python and the Pandas library.

## Dataset
- Total Records: 100
- Total Columns: 7

### Columns
- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

## Analysis Performed
- Loaded CSV using Pandas
- Checked dataset information
- Handled missing values
- Calculated Total Revenue
- Calculated Average Sales
- Identified Best Selling Product
- Generated Product-wise Sales Summary

- Average Sales: Generated by Python
- Best Selling Product: Generated by Python
